## SQL Injection Simulation Lab

This script simulates common SQL injection payloads sent to a vulnerable login endpoint.

### Objectives

- Generate malicious authentication telemetry
- Train SIEM detection logic
- Enable threat hunting pattern recognition
- Test automated response workflows

### Example Payloads

- Authentication bypass (`' OR 1=1--`)
- UNION data extraction
- Time-based injection
- Boolean logic injection
